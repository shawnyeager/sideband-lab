The index always concentrates

Twenty search engines competed to index the web in 1999. Twenty-five years later, 91% of all web searches flow through one. ARD defined the discovery layer for AI agents in June 2026. The same clock started.

Other ARD members

The amber bar marks the chokepoints: cloud runtime is where agents execute, and agent platforms are how users and systems reach them. When ARD launched June 17, 2026, only two founding members had products in both rows: Google and Microsoft.

The other nine concentrate in the remaining layers: model serving, developer tooling, enterprise data, hardware and networking. Real infrastructure, but none of it is the runtime. An agent running on GCP does not query a neutral registry. The registry GCP already trusts is easier to reach and harder to displace. That advantage existed before the first catalog was published.
